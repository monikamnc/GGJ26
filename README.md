# GGJ26

Videojuego Global GameJam 2026



Este videojuego ha sido creado por [Marc Benito](https://github.com/Kinoli7) y [Mónica Martín](https://github.com/monikamnc) en el evento presencial Global Game Jam Barcelona 2026 en la UPC.



En este juego multijugador para dos personas os enfrentaréis a crear la mejor máscara medieval para enfrentaros en una justa.
Deberéis seguir un total de tres pasos para crear vuestra máscara.



* Avivar el Fuego
* Pika el hierro
* Lija tu máscara
